Plot #2: FFT

Progress report

What we have done

We have found our main topic and some datasets that we would like to use, including clean instrumental datasets and ambient noise dataset. We applied some simple filtering, frequency domain analysis, and STFT to these clean and noisy datasets. As for the deep learning algorithm, we wrote a Matlab and a Python code that randomly pairs each clean instrumental audio with a certain noise audio and mixes them (for now we used human speech noise). We have also built a preliminary CNN structure which we will continue to refine and tune for our purposes.

Challenges

Our group is new to CNNs and Machine Learning models in general, so we have been researching scholarly articles and Matlab documentation to help us. Also, one of our goals is trying to eliminate multiple possible sources of noise, meaning that we will need a lot of training data and good time management.

What we Learned

We found that when processing audio signals, turning it into a spectrogram may produce a new perspective for analysis. The spectrogram can then be used with CNN which is normally an image processing model. This shows that Image processing models can also be used for 1 dimensional signals.
